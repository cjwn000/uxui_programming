# uxui_programming repository
이 레파지토리는 UX/UI 프로그래밍 수업에서 진행한 과제와 프로젝트를 정리한 저장소입니다.

## 01. 레파지토리 구조
cjwn000/uxui_programming <br>
├─ assignments <br>
├─ projects <br>
├─ design <br>
├─ assets <br>
└─ README.md <br>


## 02. 폴더 설명
| 폴더          | 설명                      |
| ----------- | ----------------------- |
| assignments | 수업 과제 파일 & 학습 내용       |
| projects    | 개인 프로젝트   |
| design      | UX/UI 디자인 파일 및 시안       |
| assets      | 이미지, 아이콘, 폰트 등 프로젝트 리소스 |


## 03. 사용 기술
- HTML5
- CSS3
- JavaScript
- React (예정)
- Angular (예정)


## 04. 계획
- GitHub Pages를 활용하여 프로젝트 웹사이트 배포
- 개인 도메인 연결을 통해 포트폴리오 형태의 웹사이트 구축
- 모바일 환경을 고려한 반응형 웹 디자인 적용
- 모바일 사용자 경험(UX)을 고려한 UI 인터페이스 개선
- React 및 Angular 기반의 프론트엔드 프로젝트 확장
- UI 컴포넌트 및 디자인 시스템 정리
- 사용자 중심 인터랙션 및 접근성 개선 연구
